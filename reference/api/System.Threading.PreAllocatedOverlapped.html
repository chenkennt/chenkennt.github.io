PreAllocatedOverlapped(IOCompletionCallback, Object, Object)

Initializes a new instance of the System.Threading.PreAllocatedOverlapped class, specifying a delegate that is invoked when each asynchronous I/O operation is complete, a user-provided object providing context, and managed objects that serve as buffers.

Remarks

The new instance can be passed to
