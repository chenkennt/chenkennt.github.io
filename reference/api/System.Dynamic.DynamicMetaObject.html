Create(Object, Expression)

Creates a meta-object for the specified object.


Declaration
public static DynamicMetaObject Create(object value, Expression expression)
Parameters
Type Name Description
Object value

The object to get a meta-object for.
Expression expression

The expression representing this System.Dynamic.DynamicMetaObject during the dynamic binding process.
Returns
Type Description
DynamicMetaObject

If the given object implements System.Dynamic.IDynamicMetaObjectProvider and is not a remote object from outside the current AppDomain, returns the object's specific meta-object returned by System.Dynamic.IDynamicMetaObjectProvider.GetMetaObject(Expression). Otherwise a plain new meta-object with no restrictions is created and returned.
