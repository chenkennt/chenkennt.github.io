Class DynamicObject

Provides a simple class that can be inherited from to create an object with dynamic behavior at runtime. Subclasses can override the various binder methods (GetMember, SetMember, Call, etc...) to provide custom behavior that will be invoked at runtime.

If a method is not overridden then the DynamicObject does not directly support that behavior and the call site will determine how the binding should be performed.
