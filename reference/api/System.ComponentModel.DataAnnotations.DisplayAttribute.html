Class DisplayAttribute

DisplayAttribute is a general-purpose attribute to specify user-visible globalizable strings for types and members. The string properties of this class can be used either as literals or as resource identifiers into a specified System.ComponentModel.DataAnnotations.DisplayAttribute.ResourceType
