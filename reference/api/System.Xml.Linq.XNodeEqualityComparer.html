Equals(XNode, XNode)

Compares the values of two nodes.


A null node is equal to another null node but unequal to a non-null
node. Two System.Xml.Linq.XNodes of different types are never equal. Two
System.Xml.Linq.XText nodes are equal if they contain the same text. Two
System.Xml.Linq.XElement nodes are equal if they have the same tag name, the same
set of attributes with the same values, and, ignoring comments and processing
instructions, contain two equal length sequences of pairwise equal content nodes.
Two System.Xml.Linq.XDocuments are equal if their root nodes are equal. Two
System.Xml.Linq.XComment nodes are equal if they contain the same comment text.
Two System.Xml.Linq.XProcessingInstruction nodes are equal if they have the same
target and data. Two System.Xml.Linq.XDocumentType nodes are equal if the have the
same name, public id, system id, and internal subset.
