Class ImmutableSortedSet<T>.Builder

A sorted set that mutates with little or no memory allocations, can produce and/or build on immutable sorted set instances very efficiently.

Inheritance
Object
System.Collections.Immutable.ImmutableSortedSet<T>.Builder
Namespace:System.Collections.Immutable
Assembly:System.Collections.Immutable.dll
Syntax
public sealed class Builder : ISortKeyCollection, IReadOnlyCollection, IEnumerable, ISet, ICollection, IEnumerable
Remarks
While Union(IEnumerable and other bulk change methods already provide fast bulk change operations on the collection, this class allows multiple combinations of changes to be made to a set with equal efficiency. Instance members of this class are not thread-safe.
