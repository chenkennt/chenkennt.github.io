ExceptionHandler(Int32, Int32, Int32, Int32, Int32, ExceptionHandlingClauseOptions, Int32)

Creates a description of an exception handler.

Declaration
public ExceptionHandler(int tryOffset, int tryLength, int filterOffset, int handlerOffset, int handlerLength, ExceptionHandlingClauseOptions kind, int exceptionTypeToken)
Parameters
Type Name Description
Int32 tryOffset

The offset of the first instruction protected by this handler.
Int32 tryLength

The number of bytes protected by this handler.
Int32 filterOffset

The filter code begins at the specified offset and ends at the first instruction of the handler block. Specify 0 if not applicable (this is not a filter handler).
Int32 handlerOffset

The offset of the first instruction of this handler.
Int32 handlerLength

The number of bytes of the handler.
ExceptionHandlingClauseOptions kind

The kind of handler, the handler might be a catch handler, filter handler, fault handler, or finally handler.
Int32 exceptionTypeToken

The token of the exception type handled by this handler. Specify 0 if not applicable (this is finally handler).
