Class UnobservedTaskExceptionEventArgs

Provides data for the event that is raised when a faulted System.Threading.Tasks.Task's
exception goes unobserved.

public class UnobservedTaskExceptionEventArgs : EventArgs
remarks

The Exception property is used to examine the exception without marking it
as observed, whereas the System.Threading.Tasks.UnobservedTaskExceptionEventArgs.SetObserved method is used to mark the exception
as observed. Marking the exception as observed prevents it from triggering exception escalation policy
which, by default, terminates the process.
