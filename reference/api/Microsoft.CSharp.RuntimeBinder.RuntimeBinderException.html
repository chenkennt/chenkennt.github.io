Class RuntimeBinderException

Represents an error that occurs while processing a dynamic bind in the C# runtime binder. Exceptions of this type differ from in that


represents a failure to bind in the sense of a usual compiler error, whereas
represents a malfunctioning of the runtime binder itself.
