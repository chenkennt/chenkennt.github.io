Class ImmutableDictionary<TKey, TValue>.Builder

A dictionary that mutates with little or no memory allocations, can produce and/or build on immutable dictionary instances very efficiently.

Inheritance
Object
System.Collections.Immutable.ImmutableDictionary<TKey, TValue>.Builder
Namespace:System.Collections.Immutable
Assembly:System.Collections.Immutable.dll
Syntax
public sealed class Builder : IDictionary, ICollection<>>, IReadOnlyDictionary, IReadOnlyCollection<>>, IEnumerable<>>, IDictionary, ICollection, IEnumerable
Remarks
While and other bulk change methods already provide fast bulk change operations on the collection, this class allows multiple combinations of changes to be made to a set with equal efficiency. Instance members of this class are not thread-safe.
