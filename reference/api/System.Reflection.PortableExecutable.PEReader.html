Class PEReader

Portable Executable format reader.

Inheritance
Object
System.Reflection.PortableExecutable.PEReader
Namespace:System.Reflection.PortableExecutable
Assembly:System.Reflection.Metadata.dll
Syntax
public sealed class PEReader : IDisposable
Remarks

The implementation is thread-safe, that is multiple threads can read data from the reader in parallel. Disposal of the reader is not thread-safe (see System.Reflection.PortableExecutable.PEReader.Dispose).
