TryValidateObject(Object, ValidationContext, ICollection<ValidationResult>)

Tests whether the given object instance is valid.

Remarks

This method evaluates all ValidationAttributes attached to the object instance's type. It also checks to ensure all properties marked with RequiredAttribute are set. It does not validate the property values of the object.

If is null, then execution will abort upon the first validation failure. If is non-null, then all validation attributes will be evaluated.

Tests whether the given object instance is valid.

Remarks

This method evaluates all ValidationAttributes attached to the object instance's type. It also checks to ensure all properties marked with RequiredAttribute are set. If

is true, this method will also evaluate the ValidationAttributes for all the immediate properties of this object. This process is not recursive.

TryValidateProperty(Object, ValidationContext, ICollection<ValidationResult>)

Tests whether the given property value is valid.

Remarks

This method will test each ValidationAttribute associated with the property identified by . If is non-null, this method will add a ValidationResult to it for each validation failure.

If there is a RequiredAttribute found on the property, it will be evaluated before all other validation attributes. If the required validator fails then validation will abort, adding that single failure into the when applicable, returning a value of false. If is null and there isn't a RequiredAttribute failure, then all validators will be evaluated.
