Class ImmutableList<T>.Builder

A list that mutates with little or no memory allocations, can produce and/or build on immutable list instances very efficiently.

Inheritance
Object
System.Collections.Immutable.ImmutableList<T>.Builder
Namespace:System.Collections.Immutable
Assembly:System.Collections.Immutable.dll
Syntax
public sealed class Builder : IList, ICollection, IList, ICollection, IOrderedCollection, IImmutableListQueries, IReadOnlyList, IReadOnlyCollection, IEnumerable, IEnumerable
Remarks
While AddRange(IEnumerable and other bulk change methods already provide fast bulk change operations on the collection, this class allows multiple combinations of changes to be made to a set with equal efficiency. Instance members of this class are not thread-safe.
