Class FormattableString

A composite format string along with the arguments to be formatted. An instance of this type may result from the use of the C# or VB language primitive "interpolated string".

Invariant(FormattableString)

Format the given object in the invariant culture. This static method may be imported in C# by using static System.FormattableString; . Within the scope of that import directive an interpolated string may be formatted in the invariant culture by writing, for example, Invariant($"{{ lat = {latitude}; lon = {longitude} }}")
