Class EventAttribute

Any instance methods in a class that subclasses EventSource and that return void are assumed by default to be methods that generate an ETW event. Enough information can be deduced from the name of the method and its signature to generate basic schema information for the event. The EventAttribute class allows you to specify additional event schema information for an event if desired.
