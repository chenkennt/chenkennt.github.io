Interface IProducerConsumerCollection<T>

Defines methods to manipulate thread-safe collections intended for producer/consumer usage.

Namespace:System.Collections.Concurrent
Assembly:mscorlib.dll
Syntax
public interface IProducerConsumerCollection : IEnumerable, ICollection, IEnumerable
Remarks

All implementations of this interface must enable all members of this interface to be used concurrently from multiple threads.
