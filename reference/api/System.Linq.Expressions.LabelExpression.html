Class LabelExpression

Represents a label, which can be placed in any System.Linq.Expressions.Expression context. If it is jumped to, it will get the value provided by the corresponding System.Linq.Expressions.GotoExpression. Otherwise, it gets the value in System.Linq.Expressions.LabelExpression.DefaultValue. If the System.Linq.Expressions.LabelExpression.Type equals System.Void, no value should be provided.
