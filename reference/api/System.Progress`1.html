Class Progress<T>

Provides an IProgress{T} that invokes callbacks for each reported progress value.

Inheritance
Object
System.Progress<T>
Namespace:System
Assembly:mscorlib.dll
Syntax
public class Progress : IProgress
Remarks

Any handler provided to the constructor or event handlers registered with the ProgressChanged event are invoked through a SynchronizationContext instance captured when the instance is constructed. If there is no current SynchronizationContext at the time of construction, the callbacks will be invoked on the ThreadPool.

Action<T> handler

A handler to invoke for each reported progress value. This handler will be invoked in addition to any delegates registered with the ProgressChanged event. Depending on the SynchronizationContext instance captured by the at construction, it's possible that this handler instance could be invoked concurrently with itself.
